# Taskify

Project by Abbas Harb

## The user is expected to be able to:

1. Create an account
2. Login with that account
3. Choose a theme light or dark (theme should carry on between pages)
4. Create new projects
5. Delete projects
6. View projects
7. Create tasks
8. Delete tasks
9. View tasks

## Stacks

1. HTML
2. CSS
3. Javascript
4. Axios
5. Laravel
6. JWT
